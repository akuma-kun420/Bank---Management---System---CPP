A menu-driven Bank Management System developed in C++ using Object-Oriented Programming (OOP) and File Handling.

##Features:
- Create New Bank Account
- Display Account Records
- Deposit Money
- Withdraw Money
- Check Account Balance
- Persistent Data Storage using File Handling

##Technologies Used:
- C++
- Object-Oriented Programming (OOP)
- File Handling (fstream)

##Concepts Implemented:
- Classes and Objects
- Functions
- Loops
- Conditional Statements
- File Operations
- Data Management
